HTML Tags Description List

Tables
The table tag is used to create a table in HTML, which organizes data into rows and columns.
Forms
Forms provide the backbone to any website with communication between client and server.
Division
Div is used to wrap around multiple child elements. Div divides the content into logical groups which will show the content on a separate line. This element is the direct children of dl tag if need.
Span
Span is used for single elements or short piece of content. Span shows the content inline with the rest of the content. This element is the direct children of dl tag if need.
Blockquote
Blockquote is a large, stand alone quote from another source.
